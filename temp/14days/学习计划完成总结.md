## 🎨 架构设计

学习计划教你的3层架构：

```
API层 (Routes)
    ↓
业务层 (Services)
    ↓
数据层 (Repository/ORM)
    ↓
数据库 (PostgreSQL)
```

辅助系统：
- 🔐 认证：JWT + RBAC
- 🚀 异步：Celery + Redis
- 📡 实时：WebSocket
- 📊 监控：Prometheus + 日志
- ✅ 测试：Pytest + Coverage
- 🐳 部署：Docker + Compose
