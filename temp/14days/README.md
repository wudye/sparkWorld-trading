# Qbot FastAPI后端 14天学习计划 - 完整目录

## 📋 计划概述

这是一个为期14天的综合学习计划，旨在帮助你使用**FastAPI + uv**完全重写Qbot的后端系统。从基础概念到生产级部署，循序渐进。

## 🎯 学习路径

### 第1-2周：基础架构与项目结构

#### [Day 1: FastAPI基础与项目初始化](Day01_FastAPI基础与项目初始化.md)

- **核心内容**：

  - 基础路由和参数处理
- **学习时间**：4小时
- **关键代码**：
  ```python

  app = FastAPI()
  @app.get("/api/v1/stocks/{symbol}")
  async def get_stock(symbol: str):
      return {"symbol": symbol}
  ```

## 📊 技术栈总览

| 分类 | 技术 | 用途 |
|------|------|------|
| **框架** | FastAPI | Web框架 |
| **包管理** | uv | Python包管理 |
| **数据库** | PostgreSQL | 主数据库 |
| **ORM** | SQLAlchemy | 对象关系映射 |
| **缓存** | Redis | 缓存和消息队列 |
| **任务队列** | Celery | 异步任务处理 |
| **异步** | asyncio | 异步编程 |
| **WebSocket** | Starlette | 实时通信 |
| **认证** | JWT/Bcrypt | 安全认证 |
| **测试** | pytest | 单元和集成测试 |
| **容器** | Docker | 容器化部署 |
| **监控** | Prometheus | 性能监控 |
| **日志** | python-json-logger | 日志管理 |

## ⚠️ 常见问题解答

### Q: 没有程序员基础可以学吗？
A: 需要至少了解Python基础。建议先学习Python基础再开始。

### Q: 14天能否完成所有内容？
A: 可以完成基础框架。复杂功能（如完整回测引擎）可能需要更多时间优化。

### Q: 需要Linux环境吗？
A: 不需要。Windows、macOS都支持。Docker可以确保环境一致性。

### Q: 学完后能做什么？
A: 可以构建完整的量化交易系统后端、创建其他金融应用API等。

### Q: 需要多少存储空间？
A: 约5GB（含虚拟环境和依赖）。
